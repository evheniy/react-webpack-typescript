# The Project

## Installation

To install node.js using NVM run:

```bash
nvm install
nvm use
```

Then install all dependencies:

```bash
npm ci
```

## Using

### Development

To run the project in development mode:

```bash
npm start
```

### Production

To build the project for production:

```bash
npm run build
```

### Testing

To run tests:

```bash
npm test
```
